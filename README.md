# Escape-Room-Game

# Overview

This project is a simple game where the player must solve a series of puzzles to escape a locked room. 


# Features
 - Multiple puzzle levels
 - User input / buttons
 - Different types of challenges



# How to Play
- Start the game
- Solve the puzzle for each level
- Enter answers or click the correct choices
- Progress through all levels to escape
